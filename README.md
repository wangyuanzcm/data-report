# data-report
一个使用vue3实现的拖拽生成大屏的演示项目

演示图表：
1. 订单覆盖率（折线图）
2. T+3计划合理性（柱状图）
3. 生产计划执行率（表格）
4. 客户订单满足率（仪表盘）
